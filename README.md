# Каталог Хозяюшка

Статический сайт-каталог продукции ИП "Левченко" ТМ "Хозяюшка".

## Описание

Этот сайт представляет собой каталог мясных продуктов с возможностью поиска и фильтрации по категориям. Сайт полностью статический, без серверной части, и подходит для размещения на GitHub Pages.

## Функции

- Просмотр каталога товаров
- Поиск по названию и описанию
- Фильтрация по категориям
- Переключение между светлой и темной темой
- Просмотр изображений в модальном окне
- Анимации при скролле

## Структура файлов

- `index.html` - Главная страница
- `script.js` - JavaScript логика
- `style.css` - Стили и анимации
- `data/products.json` - Данные о товарах
- `uploads/` - Изображения товаров

## Размещение на GitHub Pages

1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы в корень репозитория
3. В настройках репозитория включите GitHub Pages, выбрав источник "main branch"
4. Сайт будет доступен по адресу `https://ваш-username.github.io/название-репозитория/`

## Локальный запуск

Для локального тестирования запустите:

```bash
python -m http.server 8000
```

Затем откройте http://localhost:8000 в браузере.
